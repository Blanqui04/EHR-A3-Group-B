% Abstract
\parbox{0.9\textwidth}{
\small\textbf{Abstract:} This study analyzes sepsis epidemiology and in-hospital mortality among ICU admissions using the MIMIC-III database. We identified 7,261 sepsis cases (13.59\% prevalence) among 53,432 ICU admissions. Patients with sepsis showed 3.6-fold higher mortality (32.57\% vs 9.04\%). A logistic regression model using demographic variables achieved an AUC of 0.622, indicating that clinical variables are necessary for accurate mortality prediction. These findings underscore the substantial burden of sepsis in critical care and the need for comprehensive risk stratification tools.
}
